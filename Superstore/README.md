📊 Excel Operations for Data Analysis

**A hands-on Excel project covering essential data analysis operations — from data cleaning and transformation to building interactive sales dashboards.**

✅ Topics Covered

1. 🔤 Text to Columns
- Converted raw text data in a single column into a structured, multi-column table
- Used for splitting delimited data (comma, space, fixed-width) into usable formats

2. 🧹 Remove Duplicates
- Identified and removed duplicate records from the dataset
- Ensured data integrity before performing any analysis

3. 📅 Change Date Formats
- Converted text-based date values into proper short date format
- Standardized date fields for accurate time-based analysis

4. 🧮 Formulas
- Built two calculated columns using Excel formulas
- Applied arithmetic and logical functions to derive insights from raw data

5. 🔍 VLOOKUPs
- Used VLOOKUP to fetch and bring in the Product Manager table from a separate reference sheet
- Linked datasets dynamically without manual copy-pasting

6. 📋 Pivot Tables
- Created Pivot Tables to aggregate and summarize large datasets
- Grouped data by categories to identify patterns and trends

7. 📈 Pivot Charts
- Visualized Pivot Table data using Bar Charts, Line Charts, and Pie Charts
- Selected appropriate chart types based on the nature of the data

8. 📊 Dashboards
- Built an interactive Sales Dashboard with slicers and filters
- Combined charts, KPIs, and tables into a single-view reporting layout

💡 Key Takeaways
- Cleaning data (removing duplicates, fixing date formats, splitting columns) is the foundation of any analysis
- VLOOKUPs are powerful for joining data across multiple sheets without a database
- Pivot Tables + Charts together make it easy to summarize and communicate data visually
- Dashboards bring everything together into a single, interactive view for decision-making
